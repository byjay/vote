투표 시스템 사용 가이드

설문 생성부터 회원 등록, 투표 진행, 결과 확인까지 — 관리자를 위한 전체 운영 가이드입니다.

1. 투표 참여 방법 (회원)

회원은 아래 순서대로 진행하면 됩니다. 구글 계정만 있으면 됩니다.

①
투표 링크 접속
관리자가 보낸 링크를 클릭하거나, 주소창에 직접 입력
②
구글 로그인
[Google 계정으로 로그인] 버튼 클릭
③
이름 입력
등록된 본인의 실명(이름)을 입력
④
안건 응답 후 제출
모든 안건에 응답 후 [투표 제출] 클릭
💡 동명이인이 있는 경우:
같은 이름의 회원이 2명 이상 등록되어 있으면, 이름 입력 후 전화번호를 추가로 입력하는 단계가 자동으로 표시됩니다.
💡 다시 투표하기:
이미 투표한 경우 결과 화면에서 [다시 투표하기] 버튼을 눌러 답변을 수정하고 재제출할 수 있습니다.

2. 설문 생성 및 설정

관리자 페이지 → 설정·디자인 탭에서 투표의 기본 정보와 안건을 구성합니다.

새 설문 만들기

  1. 상단의 [+ 새 설문] 버튼 클릭 → 새 설문이 생성됩니다.
  2. 드롭다운에서 설문을 선택하여 전환할 수 있습니다.
  3. 필요 없는 설문은 [삭제] 버튼으로 제거합니다.

제목 / 부제 설정

  • 제목: 투표 화면 상단에 크게 표시되는 이름 (예: 정기총회 투표)
  • 부제 / 안내문: 제목 아래에 표시되는 설명 문구

안건(문항) 등록

  • 개별 추가: [+ 문항 추가] 클릭 → 질문 텍스트 입력
  • 일괄 추가: [일괄 추가] 클릭 → 한 줄에 안건 하나씩 입력 → 한번에 등록

안건 유형

  • 찬반 투표: 가/부 두 가지 선택 (기본값)
  • 객관식 (직접 입력): 선택지를 직접 작성 (예: 소고기, 회, 삼겹살)
  • 4지선다 / 5지선다: 1~4 또는 1~5 번호형 객관식
💡 추가 질문(조건부 문항):
각 안건의 [+ 추가질문] 버튼으로, 특정 답변 선택 시에만 나타나는 하위 질문을 연결할 수 있습니다.

3. 회원 명단 등록

관리자 페이지 → 회원 탭에서 투표 참여 대상자를 등록합니다.

입력 형식

텍스트 입력란에 한 줄에 회원 한 명씩 입력합니다. 아래 형식을 모두 지원합니다:

  • 이름 — 이름만 (가장 간단)
  • 이름 전화번호 — 동명이인 구분 시 필요
  • 이름 이메일 전화번호 — 발송 기능 활용 시
💡 엑셀에서 복사·붙여넣기:
엑셀에서 이름/전화번호 열을 선택 후 Ctrl+C → 입력란에 Ctrl+V하면 자동으로 파싱됩니다. (구분자: 공백 또는 탭)

등록 모드

  • 회원 등록 (중복제외 추가): 기존 명단 유지, 새 회원만 추가
  • 회원 일괄 덮어쓰기: 기존 명단 전체 삭제 후 새 목록으로 교체 (투표 기록도 초기화됨)
⚠️ 덮어쓰기 주의: 회원 일괄 덮어쓰기를 실행하면 기존의 모든 투표 기록이 삭제되어 복구할 수 없습니다.

4. 투표 링크 발송

관리자 페이지 → 발송 탭에서 회원별 개인 투표 링크가 담긴 메시지를 전송합니다.

메시지 템플릿 변수

  • {name} — 회원 이름으로 자동 치환
  • {link} — 해당 회원 전용 투표 링크
  • {phone} — 등록된 전화번호

발송 방법

  • 개별 복사: 각 회원 카드의 [복사] 버튼 클릭
  • 카카오톡 전송: [카카오 전송] 클릭 → 카카오 친구/대화방 선택 팝업
  • 전체 복사 / TXT 다운로드: 전체 메시지를 한 번에 복사하거나 파일로 저장 후 외부 발송 서비스 활용

5. 실시간 결과 확인

관리자 페이지 → 결과 탭에서 집계 현황을 실시간으로 모니터링합니다.

  • 투표 현황: 현재 제출 수 / 전체 회원 수를 상단에 표시
  • 자동 새로고침: [10초 자동 새로고침] 체크 시 화면 자동 갱신
  • 안건별 그래프: 각 안건의 선택지별 득표 수를 가로형 막대그래프로 표시
  • 집계 초기화: 회원 명단은 유지한 채 투표 기록만 삭제하여 재투표 진행 가능
💡 일반 회원의 결과 보기:
투표 완료 화면에서 [결과 보기] 버튼을 누르면, 개인 정보 없이 집계 결과만 익명으로 확인할 수 있습니다.

6. 관리자 페이지 접속

  • 투표 화면 우하단 톱니바퀴(⚙) 버튼 클릭 → 관리자 페이지 이동
  • 또는 URL에 ?admin=1을 붙이면 바로 접근 가능
  • 관리자 계정(designssir@gmail.com 또는 이름이 천세종인 구글 계정)으로 로그인하면 자동 인식됩니다
💡 수퍼관리자:
designssir@gmail.com 계정은 별도의 이름 입력 없이 구글 로그인만으로 관리자 권한이 자동 부여됩니다.

7. 로그아웃

투표가 끝난 후 또는 다른 계정으로 전환하려면 로그아웃합니다.

일반 회원 (투표자)

  • 투표 완료 화면 하단의 [로그아웃] 버튼 클릭
  • 또는 브라우저를 닫거나 페이지를 새로고침하면 자동으로 로그인 화면으로 돌아갑니다

관리자

  • 관리자 페이지 우상단 [로그아웃] 버튼 클릭
  • 로그아웃 후 다시 구글 계정으로 로그인해야 합니다
💡 자동 로그아웃:
보안을 위해 페이지를 새로고침하거나 재접속하면 이전 로그인 정보가 자동으로 초기화됩니다. 매번 새로 구글 로그인이 필요합니다.
